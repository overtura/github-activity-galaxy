# GitHub Activity Galaxy 자가 개선 Backlog

민감 정보 없는 샘플 데이터와 밝은 운영 지도 컨셉을 유지한다.

## Planning
- GitHub API 연결 전 필요한 데이터 계약과 privacy boundary를 문서화한다.
- activity type별 색상 의미와 legend 확장 기준을 `DESIGN.md`에 보강한다.

## Feature
- 샘플 activity bucket을 더 설명적인 구조로 정리한다.
- 기간 선택 상태에 따라 설명 문구나 metrics helper text가 바뀌도록 작게 개선한다.

## Design / Visual
- 밝은 canvas 위에서 노드와 orbit line 가시성을 개선한다.
- 모바일 metrics와 canvas 간 간격을 다듬는다.

## Accessibility
- 3D activity map의 대체 설명과 metrics summary를 보강한다.
- color-only status가 되지 않도록 label/텍스트 설명을 추가한다.

## Workflow
- self-improve 루프 결과 PR body가 GitHub 활동 지도 목적을 더 잘 설명하도록 개선한다.
