# Design References

## Reference A
- Source/product: operational data visualization tools
- Why selected: 활동 밀도와 상태를 빠르게 읽는 구조를 참고한다.
- Principles to adopt: scan-first metrics, clear status colors, compact controls
- Elements not to copy: 특정 제품 layout, brand colors, logos
- Conflicts with our product: 이 repo는 3D spatial map이므로 일반 dashboard density를 그대로 가져오지 않는다.

## Reference B
- Source/product: astronomy-inspired visual explorers
- Why selected: orbit, density, depth cue를 product signal로 쓰기 위해 참고한다.
- Principles to adopt: spatial grouping, depth contrast, readable labels
- Elements not to copy: decorative starfield-only composition
- Conflicts with our product: 활동 데이터 해석이 장식보다 우선한다.
